# Linked Events Savoy Updater

This project integrates with the City-of-Helsinki Linked Events API to fetch and update events at Savoy Teatteri in an Excel file in real-time.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python main.py`

## Requirements

- Python 3.8+
- Access to Linked Events API
- Excel file for events
